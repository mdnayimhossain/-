A frontend-only Integrated Education Management System (IEMS) built using **HTML, CSS, and JavaScript**.

## Roles
- Student
- Teacher
- Staff (Admin)

## Features
### Authentication
- Signup (Name, ID, Phone, Email, Password, Role)
- Login & Logout
- Role-based dashboard

### Student
- Registered Courses
- Attendance
- Semester Result

### Teacher
- Assigned Courses
- Attendance Management
- Marks Upload (UI)

### Staff
- Student Records
- Payments (UI)
- Reports

## Technologies Used
- HTML5
- CSS3
- JavaScript (LocalStorage)
- VS Code

## How to Run
1. Clone the repository
2. Open `index.html` using Live Server
3. Signup and login with any role

## Note
This project does not use a backend. All data is stored in browser LocalStorage.
